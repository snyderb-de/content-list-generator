# Install And Run

See the root install guide for the canonical version:

- [INSTALL.md](../INSTALL.md)

Quick summary:

- macOS and Linux use the Go GUI
- Windows uses the Python GUI with a Desktop `.bat` launcher and `customtkinter`
- alternate Go Windows `.exe` artifacts are built into `releases/windows-go/`
- local build outputs live in `build/`
- release outputs live in `releases/`
